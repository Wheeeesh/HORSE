# TADAA (Vite + React) — GitHub Pages setup (no terminal)

This repository is a Vite/React web app. It can be deployed to GitHub Pages automatically using GitHub Actions.

## One-time GitHub setup

1. Upload these files to a new GitHub repository (make sure `package.json` is in the repository root).
2. In the repository: **Settings → Pages → Source → GitHub Actions**
3. Go to the **Actions** tab and wait for the "Deploy to GitHub Pages" workflow to finish.

After it succeeds, your site URL will appear in **Settings → Pages**.

## Editing in the browser

You can edit files directly on GitHub:
- Open a file → click the pencil icon → edit → commit.
Each commit triggers a new deploy.
